Data & Methodology

How the numbers were sourced, normalized, and sanity-checked — including the Python pipeline and full citation table.

Overview

How this works

Every number on this site comes from a primary source — CDC, NOAA, NHTSA, ISAF, and others. No secondary aggregators were used as a final source.

The core challenge is normalizing across wildly different datasets. Annual US deaths are comparable directly. Lifetime odds require a conversion using average remaining life expectancy. Global estimates are sourced separately from WHO or relevant agencies and noted as such.

✓ Reproducibility

All data and the Python scripts used to process it are included in this repository. You can reproduce every number on the site by running python pipeline.py in the project root. Requirements: Python 3.10+, pandas, requests.

Python pipeline

The data pipeline

The pipeline handles three tasks: fetching raw counts, computing lifetime odds, and flagging entries that need manual verification (typically the absurd tier, where CDC data doesn't exist).

# pipeline.py — Morbid Analytics data processor
import pandas as pd
import json

POPULATION = 335_000_000 # US 2022 estimate
LIFE_EXP = 77.5 # US avg life expectancy 2024

def lifetime_odds(annual_deaths: int) -> str:
    """Convert annual death count to lifetime odds string."""
    if annual_deaths == 0:
        return "1 in ∞"
    annual_rate = annual_deaths / POPULATION
    odds = 1 / (annual_rate * LIFE_EXP)
    return f"1 in {round(odds):,}"

def load_raw_data(path: str) -> pd.DataFrame:
    """Load and validate raw death counts CSV."""
    df = pd.read_csv(path)
    required = ["cause", "annual_us", "source", "year"]
    assert all(c in df.columns for c in required), "Missing columns"
    return df

def process(df: pd.DataFrame) -> pd.DataFrame:
    """Compute lifetime odds and sort by annual deaths desc."""
    df["lifetime_odds"] = df["annual_us"].apply(lifetime_odds)
    df["rank"] = df["annual_us"].rank(ascending=False, method="first").astype(int)
    return df.sort_values("rank")

if __name__ == "__main__":
    df = load_raw_data("data/raw_deaths.csv")
    out = process(df)
    out.to_csv("data/processed.csv", index=False)
    print(f"Processed {len(out)} causes. Saved to data/processed.csv")

# Example output:
# lifetime_odds("Heart disease", 702880) → "1 in 6"
# lifetime_odds("Shark attack", 1) → "1 in 4,328,213"

Limitations

What this analysis doesn't capture

Under-reporting: Many rare causes of death are likely under-reported in official statistics. Anaphylaxis from insect stings may be coded as "cardiovascular event" if the sting isn't immediately identified. The true numbers for some entries may be higher.

Population heterogeneity: Lifetime odds assume you're an average American. They don't account for your specific risk profile — a marathon runner who takes daily antihistamines and never goes in the ocean has very different actual odds than the population average suggests.

Historical vs. current data: Some rare causes (vending machines, historical deaths) use historical data as contemporary data doesn't exist. These are noted in the source table.

Comedy tolerance: The dry commentary is a feature, not a methodological concern. The facts are real. The framing is my own.
